# T-Goda Hotel Booking - Design Context

## Users
- **Primary**: นักท่องเที่ยวทุกช่วงวัย - ตั้งแต่ Gen Z ถึงผู้สูงอายุ
- **Context**: ค้นหาและจองโรงแรมผ่านมือถือ/คอมพิวเตอร์ ทั้งในประเทศและต่างประเทศ
- **Job to be done**: หาโรงแรมราคาดี จองง่าย มั่นใจได้

## Brand Personality
- **3 Words**: สดใส (Vibrant), น่าเชื่อถือ (Trustworthy), เป็นมิตร (Friendly)
- **Tone**: อุ่นเป็นมิตรแต่ยังมืออาชีพ ไม่ดูแพงเกินเข้าถึงยาก
- **Emotional goals**: ความมั่นใจในการจอง + ความตื่นเต้นในการเที่ยว

## Aesthetic Direction
- **Visual tone**: Clean, organized, approachable luxury (ไม่หรูเกินไป)
- **Reference**: SVG designs from KU Competition – Web2026 (Home Page.svg, Search Result.svg, Room Detail.svg)
- **Theme**: Light mode สดใส พื้นหลังขาว/เทาอ่อน เน้น accent colors ที่สด
- **Typography**: Plus Jakarta Sans - ทันสมัย อ่านง่าย รองรับทุกวัย

## Design Principles
1. **Fidelity to original** - ทำให้ตรง SVG reference มากที่สุด ทุก pixel มีความหมาย
2. **Universal accessibility** - ขนาดตัวอักษรใหญ่พอ คอนทราสต์ชัด ใช้งานง่ายทุกวัย
3. **Trust through clarity** - ข้อมูลราคา รีวิว รูปภาพ ต้องชัดเจน ไม่ซ่อนไว้
4. **Warmth without clutter** - ใช้สี accent ที่อบอุ่นแต่ไม่รกตา
5. **Responsive by default** - ทุก screen size ต้องใช้งานได้ดี

## Technical Constraints
- React + Vite
- CSS Variables for design tokens
- No external UI libraries (custom components)
- Mobile-first responsive
